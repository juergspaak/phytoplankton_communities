This folder contains all programms and data external data used in connection with the paper

"Effects of pigment richness and size variation on coexistence, richness and function in light limited phytoplankton"
by J.W. Spaak and F. DeLaender, 2021


##########################################################################
Python code:
##########################################################################


generate_species.py
	Contains functions to generate absorption spectra and phtosynthetic efficiencies

I_in_functions.py
	Load incoming light intensities and fluctuating light spectra

richness_computation.py
	Computes equilibrium densities of communities

##########################################################################
Data
##########################################################################

background_absorption.csv
	Contains the absorption spectrum of sea water
	as well as background absorption from dissolved particles
	data taken from Stomp et al. 2007
sunlight.csv
	Contains light spectrum of incoming light intensity,
	data taken from Stomp et al. 2007
pigments.csv
	Absorption spectrum of the used pigments
	data sources mentioned in the main paper
Pigment_algae_table.csv
	Which pigmentation type contains which pigments at which concentration
